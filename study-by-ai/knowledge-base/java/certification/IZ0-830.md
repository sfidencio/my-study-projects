### 游릭 Q1. What is the output of the following code?
```java
int x = 5;
System.out.println(++x);
```
<details>
<summary>Show Answer</summary>

**Explanation:**  
The `++x` is the pre-increment operator, which increments `x` before its value is used in the expression.  
So, `x` becomes 6, and then 6 is printed.

**Output:**  
6

**Example:**
```java
int x = 5;
System.out.println(++x); // prints 6
```
</details>

### 游릭 Q2. Which of the following are valid Java identifiers?  
a) _myVar  
b) 2ndVar  
c) $value  
d) class

<details>
<summary>Show Answer</summary>

**Explanation:**  
- Identifiers can start with a letter, underscore `_`, or dollar sign `$`.
- They cannot start with a digit.
- Reserved words (like `class`) cannot be used as identifiers.

**Valid:** a) _myVar, c) $value  
**Invalid:** b) 2ndVar (starts with digit), d) class (reserved word)

**Example:**
```java
int _myVar = 1; // valid
int $value = 2; // valid
// int 2ndVar = 3; // invalid
// int class = 4; // invalid
```
</details>
